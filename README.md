# Software Portfolio

Static portfolio page for GitHub Pages deployment.

## Files

- `index.html`
- `styles.css`
- `assets/`

## Deploy on GitHub Pages

1. Push this repository to GitHub.
2. Open the repository's `Settings` tab.
3. Go to `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the main branch and the `/ (root)` folder, then save.
6. Wait for GitHub Pages to publish the site URL.

## Update links

Replace the placeholder URLs near the top of `index.html` for:

- resume
- GitHub
- multimedia portfolio
- research links
- canonical URL
- Open Graph URL
